Animated Flip Boxes WordPress plugin highlights your content inside your page using beautiful flipbox layouts (flipboxes). Add anywhere using shortcode.
== Description ==
Use animated Flip Boxes WordPress plugin to highlight your content inside your page in a great way. Use shortcode to add anywhere. These simple awesome filp boxes will add a special look and user experience inside your webpage.

Please check demo of the Flipboxes:-

<a href="http://flipboxtest.coolplugins.net/flipboxes-demos">Click Here to view demo</a>

<h2>Special features of WordPress Flip Boxes:-</h2>
<ul>

<li>Customize a flip box look and feel with custom colors.</li>
<li> Easily add font-awesome icons with content boxes.</li>

== Installation ==
<h6>There are 3 ways to install Flip Boxes inside your wordpress website.</h6>
<ol>
<li><strong>Automatically install our plugin via WordPress admin panel</strong>:- Open your WordPress website admin panel and go to Plugins > Click Add New & search '<strong>Flip Boxes</strong>' > Here you will find our plugin > Now click on install button > After this you will see a activate button > Just activate it to use Flip Boxes inside your website</li>

<li><strong>Install via Zip file</strong>:- In this method first you need to download our plugin from wordpress plugins directory. Go to https://wordpress.org/plugins/flip-boxes <strong>Flip Boxes</strong>. After this inside your WP-admin panel, click on Plugins > Add New > Upload Button. Now choose the plugin zip file that you just downloaded from wordpress plugins directory & activate it.</li>

<li><strong>Install plugin via FTP</strong>:- In this method you also first need to download plugin zip file from wordpress plugins directory. After this open your FTP manager and Go to > wp-content/plugins folder. Here you need to upload extracted version of <strong>flip-boxes</strong> (please remember don't upload zip file directly here) After this you can activate plugin from wp-admin panel plugins page.</li>
</ol>
